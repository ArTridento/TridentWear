Brand Story

Clean clothing with a clear product focus.

TridentWear is built around well-structured T-shirts, readable product details, and a catalog that is easy to browse.

Origin

Designed for daily rotation

TridentWear started from a simple idea: a clothing site should feel clean, modern, and easy to use while still showing enough product depth to make every T-shirt easier to understand. Fabric, fit, GSM, and size details are treated as core content instead of hidden extras.

All 18 T-shirt categories are visible from the shop page.
Built local-first with FastAPI, HTML, CSS, and JS.
Product pages show fabric, fit, neck type, GSM, and size chart details.
Catalog

Clear category structure

Fit type, neck type, fabric type, design type, and special type all work together in one filter system.

Clarity

Focused product details

Every product page explains material, GSM, fit, neck type, and size selection before checkout.

Control

Admin-ready storefront

Add, edit, and feature products directly from the local admin panel without breaking the layout.

What We Build

Product pages that answer real questions

Instead of a flat catalog, TridentWear shows what matters first: price, fit, material, size options, and related products that share category attributes.

Store Experience

Simple flow from browse to checkout

The shop, cart, contact, and auth pages are built to run locally with the FastAPI backend so the project is ready for demos and further deployment work.
